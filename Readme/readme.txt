=========================================================
README - SISTEMA DE PEDIDOS DE VENDA
=========================================================

📌 Sobre o projeto
---------------------------------------------------------
Este é um sistema de Pedidos de Venda desenvolvido em Delphi,
utilizando banco de dados MySQL e FireDAC para acesso.

- Linguagem: Delphi VCL
- Banco de dados: MySQL
- Framework de acesso: FireDAC
- Estrutura: Clientes, Produtos, Pedidos e Itens do Pedido

---------------------------------------------------------
⚙️ Estrutura do banco de dados
---------------------------------------------------------
O script SQL fornecido cria toda a estrutura necessária:

- clientes: cadastro de clientes
- produtos: cadastro de produtos
- pedidos: cabeçalho dos pedidos
- pedidos_produtos: itens vinculados ao pedido


---------------------------------------------------------
🖥️ Funcionalidades principais
---------------------------------------------------------
- Inclusão de pedidos com múltiplos itens
- Cálculo automático do valor total
- Consulta e exclusão de pedidos
- Interface com DBGrid para visualização dos itens
